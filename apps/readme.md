# Apps

Homebrew packages and Mac App Store apps backup/restore.

| File                  | Contents                                  | Generated by                          |
|-----------------------|-------------------------------------------|---------------------------------------|
| `brew-list.txt`       | Formulae installed on request             | `brew leaves --installed-on-request`  |
| `brew-list-cask.txt`  | Casks                                     | `brew list --cask`                    |
| `brew-tap.txt`        | Taps                                      | `brew tap`                            |
| `mas-list.txt`        | Mac App Store apps (`id  name  (ver)`)    | `mas list`                            |

## Backup everything

From the repo root, regenerate the lists above:

```shell
make apps-backup
```

## Install everything from backup

Run from the repo root (`~/.dotfiles`):

```shell
xargs -n1 -t brew tap < apps/brew-tap.txt
xargs -n1 -t brew install < apps/brew-list.txt
xargs -n1 -t brew install --cask < apps/brew-list-cask.txt
awk '{print $1}' apps/mas-list.txt | xargs -n1 -t mas install
```

> [!NOTE]
> `mas install` requires being signed in to the App Store first (and `mas`
> itself: `brew install mas`).

## Install MongoDB tools and shell only

`mongosh` is already in `brew-list.txt`; the tools below come from the
`mongodb/brew` tap and are not in the backup lists:

```shell
brew tap mongodb/brew
brew install mongodb/brew/mongodb-community-shell
brew install mongodb/brew/mongodb-database-tools
```

More info: https://www.mongodb.com/docs/manual/tutorial/install-mongodb-on-os-x/
